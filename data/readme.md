# Dataset Location

Due to GitHub file size limitations, the original dataset files are not included directly in this repository.

Please download the original dataset from:
https://www.kaggle.com/datasets/crainbramp/steam-dataset-2025-multi-modal-gaming-analytics

After downloading and extracting the archive, the files used in this project can be found at:

```
archive/steam_dataset_2025_csv_package_v1/steam_dataset_2025_csv/
```

This project uses the following files:

* `applications.csv`
* `application_genres.csv`
* `genres.csv`

After extraction, place the files into the `data/` directory.
